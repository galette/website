La Galette 0.63.3 est cuite, à table ! :-)

Cette version corrige, entre autres, une faille de sécurité mise en évidence sur la version de démonstration. Voici la liste des corrections :

  • correction d'une faille qui permettait sur certains serveurs à l'attaquant d'envoyer des fichiers PHP arbitraires
  • lors de l'envoi d'un formulaire de création/modification d'adhérent invalide, les champs dynamiques de type ligne étaient répétés bogue #10187
  • des problèmes d'encodage sur les données ont été constatés sur des serveurs MySQL configurés en UTF-8. La connexion est désormais forcée en LATIN1 (merci à Cédric)
  • des espaces insécables apparaissaient en clair dans le texte des emails non html (merci à Cédric)
  • l'utilisation de caractères XML dans le sujet d'un mailing causait une erreur d'analyse XML lors de la prévisualisation bogue #14571
  • des données inutiles étaient stockées dans l'historique (et pas au bon endroit) lors de l'envoi de mailing bogue #14569
  • des erreurs d'analyse XML survenaient sur la page d'historique lorsque l'on avait saisi des caractères réservés dans une fiche adhérent (bogue #14561)
  • les balises saisies dans un mailing ne s'affichaient pas sous Firefox bogue #14465
