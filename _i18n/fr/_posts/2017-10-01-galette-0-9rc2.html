Deux semaines après la première.. Voici une seconde version de test publique pour la version 0.9 de Galette !

Un très, très grand merci à toutes les personnes qui ont testé la version ; quelques anomalies ont été remontées, et elles ont bien sûr été corrigées dans cette nouvelle mouture !

Depuis la première RC, quelques petites améliorations ont été apportées :

  • réintégration du plugin admintools,
  • possibilité de renseigner deux années dans les préférences des cartes d'adhérents,
  • possibilité de chercher dans les libellés des statuts depuis la recherche libre,
  • utilisation de listes dans la recherche libre pour les statuts et les genres.

Pour rappel...

Cette nouvelle version majeure apporte, outre la correction de quelques bogues, un certain nombre de nouvelles fonctionnalités ; que je vous invite à découvrir en testant cette version :-)

Bien qu'il s'agisse d'une version majeure, aucune modification n'a été apportée à la base de données ; il suffit donc « simplement » de mettre à jour vos fichiers !

Notez que cette nouvelle mouture rend les plugins existants incompatibles... Les plugins officiels maps, paypal, fullcard et auto ont dores et déjà été migrés ; vous pourrez les tester en récupérant le fichier ZIP de leur branche develop sur github.

N'hésitez pas à tester cette nouvelle version et me faire des retours, soit sur les listes de diffusion, soit sur le tracker :-)
