A few months after the last major release, we are proud to announce the release of Galette 0.9.1!

This new version provides some bug fixes, and a few new features:

  • batch changes on members,
  • change sender on mailings,
  • choose default status for new members from preferences,
  • a text can be added on footer,
  • German translation.

Full cangelog for Galette 0.9.1 is available at:
{{ site.galette.doc_url }}/{{ site.lang }}/master/changelog.html#v091

Along with this new release, a plugin to manage events is now available!

I'd like to thank very much all persons involed in this new release; and especially Arnold W. for the german translation, as well as "Association Bretonne des Amis de Saint Jacques de Compostelle " (https://www.compostelle-bretagne.fr/) which sponsored sender choice on mailing and the whole events plugin.
